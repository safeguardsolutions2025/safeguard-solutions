Coloca aquí tus FOTOS y VIDEOS del carrusel.

Opción 1 (recomendada): crea un archivo media.json con el orden exacto, ejemplo:
[
  { "src": "/imagenes/recepcion.jpg", "type": "image" },
  { "src": "/imagenes/auditoria.mp4", "type": "video" },
  { "src": "/imagenes/tanque.webp", "type": "image" }
]

Opción 2 (sin editar nada): nombra tus archivos como:
slide1.jpg, slide2.mp4, slide3.jpg, slide4.mp4, ... (hasta slide30.*)
Extensiones soportadas: .jpg .jpeg .png .webp .mp4 .webm .ogg .mov
